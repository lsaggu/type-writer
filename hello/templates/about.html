About type-player

type-player is designed to accept text input from a user and to create a short video clip of the inputted text being typed out. This video clip is available for the user to download and play independently - or the user might decide to use it as part of a larger video production.

type-player is a web application built using Python and Django. The front end is supported by Javascript, jQuery, and Bootstrap 4 (Bootstrap likely being obvious ). This tool is currently still under development, but there will be more to come here! We hope that type-player proves to be a fun, useful tool!

Does this look familiar?

If type-player looks familiar to something else you've seen, it's because it is! type-player is a sister application to type-recorder, created by the same author. type-recorder takes a string of text and converts it to an audio format, whereas type-player takes a string of text and converts it to a video format. Pretty cool, right!?
